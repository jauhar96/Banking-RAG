# SOP - Suspected Account Takeover (ATO)

## Indicators
- Login from an unusual device or location.
- Abnormal transaction activity within a short time window.
- Profile changes without the user's confirmation.

## Procedure
1. Verify user identity.
2. Temporarily block access if strong suspicion is detected.
3. Reset credentials and re-bind the trusted device.
4. Educate the user about phising prevention and OTP/PIN confidentiality.
5. Create an incident record for internal audit.

## Escalation
Escalate to the risk/fraud team if repeated attack patterns are detected or if the transaction value is significant.
